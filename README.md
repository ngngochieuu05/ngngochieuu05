## 🚀 Dự án nổi bật

### [Con Bò Cười](https://github.com/ngngochieuu05/Con_Bo_Cuoi) 
Xây dựng ứng dụng cross-platform (Desktop + Web) giúp giám sát đàn bò thời gian thực bằng trí tuệ nhân tạo. Hệ thống tự động phát hiện bò, nhận diện hành vi bất thường, cảnh báo dấu hiệu bệnh sớm và kết nối nông dân với chuyên gia thú y.
**Công nghệ:** `Flet 0.28.3 (Python), CV,YOLOv8 + PyTorch + OpenCV Data, JSON → PostgreSQL Security`

### [Ứng dụng hỗ trợ học Tiếng Anh](https://github.com/ngngochieuu05/App_Ho_Tro_Hoc_Tieng_Anh) 
Xây dựng một ứng dụng hỗ trợ học tiếng Anh dành cho người Việt, giúp người dùng cải thiện kỹ năng nghe – nói – đọc – viết một cách dễ dàng và tương tác.
**Công nghệ:** `Python 3.14+, Flet 0.28.3 (Desktop + Web)`

### [Hệ Thống Giám Sát Lái Xe Thông Minh](https://github.com/ngngochieuu05/He_thong_giam_sat_lai_xe) 
Hệ thống giám sát lái xe dựa trên AI, theo dõi hành vi và trạng thái người lái thời gian thực (30 FPS). Phát hiện các bất thường như mất tập trung, mệt mỏi (ngáp, nhắm mắt), sử dụng điện thoại... và đưa ra cảnh báo tức thời. Hỗ trợ đăng nhập bảo mật bằng nhận diện khuôn mặt (Face ID với ArcFace), giao diện thân thiện, dashboard quản trị, chatbot hỗ trợ và xuất báo cáo PDF/Excel. Kiến trúc chạy trên edge device (không cần database truyền thống).
**Công nghệ:** `Python 3.9+ Flet, YOLOv8 (Ultralytics), ArcFace (InsightFace), ONNX Runtime, OpenCV, NumPy scikit-learn, PyCryptodome`

### [TrafficMonitorApp - Hệ Thống Giám Sát Giao Thông Thông Minh](https://github.com/ngngochieuu05/BCCK_Nhom3_TrafficMonitorApp) 
Ứng dụng Windows Forms (.NET 6) giám sát và phân tích giao thông thời gian thực bằng AI. Hỗ trợ phát hiện, phân loại và đếm phương tiện (xe máy, ô tô, xe buýt, xe tải...) với hai chế độ Line Crossing và Zone Detection. Cung cấp thống kê realtime, biểu đồ, cảnh báo tắc nghẽn, xuất báo cáo Excel và chatbot hỗ trợ.
**Công nghệ:** `C# (.NET 6.0 - Windows Forms), OpenCvSharp4, ONNX Runtime, Microsoft.ML, YOLOv8, Entity Framework Core, SQL Server (LocalDB), LiveCharts (WPF & WinForms), Newtonsoft.Json, EPPlus System.Drawing`

### [POTATO COUNTER APP - Hệ Thống Đếm Và Phân Loại Chất Lượng Khoai Tây Thông Minh](https://github.com/ngngochieuu05/POTATO_COUTER_APP) 
Ứng dụng AI tự động đếm và phân loại chất lượng khoai tây (Tốt/Kém) trên dây chuyền sản xuất thời gian thực. Sử dụng YOLOv8 để phát hiện & theo dõi, ResNet50 để phân loại (độ chính xác ~95%). Hỗ trợ giao diện GUI, báo cáo PDF/Excel, biểu đồ thống kê, tốc độ 25-30 FPS.
**Công nghệ:** `Python 3.10+, PyTorch 2.0+, Ultralytics, YOLOv8, ResNet50 (CNN), OpenCV 4.8+ PIL/Pillow, NumPy, Pandas scikit-learn, Matplotlib / Seaborn / Plotly TKinter / PyQt5 (GUI), imgaug, tqdm, TensorBoard`
